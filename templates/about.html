项目介绍

本项目旨在通过互动的方式，帮助用户了解和学习中国传统的二十四节气文化。

技术栈

  • 后端: Python (Flask)
  • 前端: HTML, CSS, JavaScript
  • 数据存储: JSON 文件

AI 应用

项目中集成了AI技术，例如：

  • 智能问答: 在“AI互动”模块中，利用大语言模型 (LLM) 理解用户问题并提供相关的节气知识解答 (当前支持 ChatGPT, DeepSeek V3, Gemini 2.5 Pro)。
  • 个性化模式: AI 助手支持多种模式（聊天、猜节气、旅行推荐），提供更丰富的互动体验。

创新点

  • 互动体验: 通过记忆匹配游戏、诗词配对、AI问答等多种互动形式，提升学习趣味性。
  • 文化融合: 将传统文化与现代技术相结合，让古老的节气知识焕发新生。
  • 多 AI 模型集成: 提供不同 AI 模型的选择，用户可以体验不同模型的回答风格和能力。
  • 流式响应: AI 回答采用流式输出，提升用户体验。

应用点

  • 教育科普: 作为中小学或文化爱好者的节气知识学习工具。
  • 文化传播: 向更广泛的人群介绍和推广二十四节气文化。
  • 技术实践: 作为 Web 开发 (Flask)、前端交互和多 AI API 集成的实践案例。
